# Sales Performance Dashboard

This project analyses a sample sales dataset using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualisation
- Basic formulas

## Analysis Performed
- Revenue by salesperson
- Revenue by product
- Revenue by month

## Key Insights
- Samira generated the highest total revenue.
- Phones produced the highest revenue across products.
- Sales remained consistent across January and February.

This project demonstrates basic data analysis and dashboard creation using Excel.
